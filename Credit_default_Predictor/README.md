# Credit Default Prediction

## Project Overview
Predict whether a customer will default on credit card payments.

## Dataset
Taiwan Credit Card Default Dataset

## Models Used
- Logistic Regression
- Decision Tree
- Random Forest
- XGBoost

## Explainability
- SHAP Summary Plot
- SHAP Bar Plot

## Final Model
XGBoost

ROC-AUC Score: 0.778

## Deployment
Streamlit Web Application
